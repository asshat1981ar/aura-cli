            [--scaffold SCAFFOLD] [--scaffold-desc SCAFFOLD_DESC] [--evolve]
            COMMAND ...

positional arguments:
  COMMAND
    bootstrap           Create default config
    config              Show effective config
    contract-report     Print CLI contract report
    diag                MCP diagnostics snapshot
    doctor              Run system diagnostics
    evolve              Run innovation workflow
    goal                Goal queue operations
    help                Show CLI help
    logs                Stream AURA logs
    mcp                 MCP HTTP client commands
    memory              Semantic memory operations
    metrics             Show performance metrics
    queue               Goal queue management
    readiness           Check V2 runtime readiness
    sadd                Sub-Agent Driven Development
    scaffold            Scaffold a project
    studio              Launch AURA Studio
    watch               Launch TUI monitor
    workflow            Workflow operations

options:
  -h, --help            show this help message and exit
  --version             show program's version number and exit
  --json-help           Emit machine-readable CLI help schema.
  --json                Output machine-readable JSON where supported.
  --dry-run             Do not apply filesystem changes.
  --decompose           Enable task decomposition mode.
  --model MODEL         Override model name for this run.
  --anthropic-api-key ANTHROPIC_API_KEY
                        Override Anthropic API key for this run.
  --beads               Enable BEADS gating for this invocation.
  --no-beads            Disable BEADS gating for this invocation.
  --beads-required      Require BEADS to allow execution for this invocation.
  --beads-optional      Enable BEADS without making it a hard gate for this
                        invocation.
  --explain             Print decision log for one-off goal runs.
  --max-cycles MAX_CYCLES
                        Maximum loop cycles for one-off runs.
  --add-goal ADD_GOAL   Legacy: add a goal to the queue.
  --run-goals           Legacy: run queued goals.
  --status              Legacy: show goal status.
  --goal GOAL           Legacy: run one goal immediately.
  --workflow-goal WORKFLOW_GOAL
                        Legacy: workflow goal text.
  --workflow-max-cycles WORKFLOW_MAX_CYCLES
                        Legacy: workflow max cycles.
  --mcp-tools           Legacy: list MCP tools.
  --mcp-call MCP_CALL   Legacy: MCP tool name to call.
  --mcp-args MCP_ARGS   Legacy: JSON args for MCP call.
  --diag                Legacy: run MCP diagnostics.
  --bootstrap           Legacy: bootstrap config.
  --scaffold SCAFFOLD   Legacy: scaffold template name.
  --scaffold-desc SCAFFOLD_DESC
                        Legacy: scaffold description.
  --evolve              Legacy: run innovation workflow.
