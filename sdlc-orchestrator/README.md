# sdlc-orchestrator

A Botpress ADK scaffold for an SDLC (Software Development Lifecycle) orchestration bot. This bot is designed to manage software projects through their full lifecycle phases using Botpress tables, conversations, and agent integrations.

## What This Contains

This directory is a **Botpress ADK scaffold** generated by the Botpress CLI (`bp new`). It provides the runtime infrastructure for a conversational bot that tracks SDLC projects, tasks, and artifacts.

### Structure

```
sdlc-orchestrator/
├── package-lock.json          # Locked dependency versions
├── .adk/                      # Auto-generated ADK runtime files (DO NOT EDIT)
│   ├── bot/                   # Bot entry point and runtime wiring
│   │   ├── src/               # ADK-generated TypeScript sources
│   │   ├── bot.definition.ts  # Bot definition with tables, workflows, states
│   │   ├── package.json       # Bot package (private)
│   │   └── tsconfig.json      # TypeScript config for bot
│   ├── integrations/          # Integration type definitions (webchat, chat)
│   ├── interfaces/            # Interface type definitions (llm, listable, typing-indicator)
│   ├── client.ts              # Typed ADK client wrapper
│   └── *.d.ts                 # Runtime type augmentation files
```

### Data Model (Botpress Tables)

| Table | Purpose |
|-------|---------|
| `ProjectTable` | Tracks software projects with name, status, phase, metadata |
| `TaskTable` | Tasks assigned to bots, with status, priority, artifacts |
| `ArtifactTable` | Outputs produced per task (specs, code, reports) |

### Conversation State

- `projectId` tag — links conversations to a specific project
- `phase` tag — tracks current SDLC phase (ideation → maintenance)
- `BotState.activeProjectId` — globally active project
- `BotState.pipelineRunning` — whether the pipeline is executing

## Installation

```bash
# Install dependencies (requires @botpress/runtime)
npm install
```

## Build

This scaffold requires the Botpress ADK CLI to build and deploy:

```bash
# Install Botpress CLI globally
npm install -g @botpress/cli

# Build the bot
bp build

# Deploy to Botpress Cloud
bp deploy
```
