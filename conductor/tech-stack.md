# AURA Technology Stack

## Core
- **Language:** Python 3.9+
- **AI/LLM:** Gemini (2.0 Flash / 2.5 Pro) via OpenRouter API
- **VCS:** Git (via GitPython)

## Frameworks & APIs
- **Web:** FastAPI & Uvicorn
- **Validation:** Pydantic
- **MCP:** Model Context Protocol (standard implementation)

## Data & Intelligence
- **Persistence:** SQLite (Local Brain & Vector Store)
- **Caching:** Momento (Distributed session memory)
- **Math/Vectors:** Numpy
- **Graphs:** NetworkX
- **NLP:** TextBlob

## Interface
- **TUI:** Rich (Terminal UI & Live Dashboard)
- **CLI:** Native argparse with subcommands

## Quality Assurance
- **Testing:** Pytest
- **Linter:** Ruff (implied by typical Python quality standards)
