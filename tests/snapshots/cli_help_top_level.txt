AURA CLI

Commands:
  help         Show CLI help
  doctor       Run system diagnostics
  readiness    Check V2 runtime readiness
  bootstrap    [EXPERIMENTAL] Create default config
  config       Show effective config
  contract-report [EXPERIMENTAL] Print CLI contract report
  diag         [EXPERIMENTAL] MCP diagnostics snapshot
  logs         Stream AURA logs
  watch        [EXPERIMENTAL] Launch TUI monitor
  studio       [EXPERIMENTAL] Launch AURA Studio
  goal         Goal queue operations
  workflow     Workflow operations
  mcp          Repo-local MCP commands
  beads        BEADS contract commands
  scaffold     Scaffold a project
  evolve       Run innovation workflow
  queue        Goal queue management
  memory       Semantic memory operations
  metrics      Show performance metrics
  sadd         [EXPERIMENTAL] Sub-Agent Driven Development
  innovate     Innovation Catalyst session management
  agent        Agent SDK commands
  credentials  Secure credential management

Examples:
  python3 main.py goal add "Refactor queue" --run
  python3 main.py mcp tools
  python3 main.py help goal add

Legacy flat flags remain supported but are deprecated.
